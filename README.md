# Intent-and-Trajectory-Prediction
Built a predictive system that goes beyond tracking—anticipating where agents will move next by combining intent classification with multi-modal trajectory forecasting. Using temporal sequence modeling and interaction-aware predictions, the system generates multiple possible future paths with associated confidence, enabling vehicles to act proactively rather than reactively in complex urban scenarios.
The UI includes a real-time visualization dashboard for agent tracking, intent inference, collision risk scoring, and uncertainty-aware trajectory prediction.
